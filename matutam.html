Matutam

Go ahead and check out matutam now!

It's a comparison of the largest search engines, Google and Bing, with some fun javascript that makes using them both side-by-side a real possibility! If you like the site and want to keep up with what's going on there, check out the blog.

It took only a couple days to build and I'll keep iterating on the features and design in the near future. Hopefully I can learn a bit more about javascript, because its power is really the future of the web.
