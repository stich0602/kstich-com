DeTAG

DeTAG is a Live Action Role-Playing (LARP) tag game designed by game students at DePaul University. Each year the students come up with a new game mechanic to add to the game, like reverse tagging and "shooting the moon." Students form equal size teams and are given the task of tagging their target team while not being tagged by the team targeting them. For a full set of this year's rules follow this link.

My job is to program all of the games features in a timely fashion. The students come up with the new feature and it has to be implemented before their session of the game begins. The application is written in PHP with a MySQL database. The PHP generates dynamic HTML tables and charts with the Google Charts API. Around 100 students play the game every session and there is an in depth statistical system in place for an awards ceremony.
